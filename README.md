# Birds I Vue

## twitter clone project to practice some vue techniques

### What I'll be using: 
#### Nuxt 3
#### Tailwind CSS
#### Prisma
#### JWT
#### Cloudinary
#### Other stuff perhaps

## Commands Run:
- npx nuxi init birds-i-vue
- npm i -D @nuxtjs/tailwindcss
- npx tailwindcss init
- npm run dev
- npm i @heroicons/vue
- npm i prisma -D
- npx prisma (test)
- npx prisma init
- npx prisma db push (after model is created)
- npm i bcrypt
- npm i jsonwebtoken
